Pasos para la ejecución del proyecto desarrollador en angular de CrytoCoins:

1.	Descargar el Git desde su página principal, link: https://git-scm.com/downloads

2.	Una vez descargado e instalado el Git, crearemos una carpeta donde se desee clonar el proyecto y entraremos a la misma.

3.	 Una vez dentro le daremos click derecho y elegiremos “Git Bash Here” y seguidamente desplegara una terminal de Git.

4.	Debemos validar nuestra identidad por lo tanto ejecutaremos “$ git config --global user.name ‘(Su nombre entre comillas)’” y 
luego se ejecutara “$ git config --global user.email su_email@example.com” (Este paso solo se ejecutara si no posee el Git configurado).

5.	Luego tendremos que copiar el siguiente link https://github.com/jhonnerg/betterplancryptocoins.git .

6.	Seguidamente ejecutaremos en el comando clonar con la url correspondiente, comando: $ git clone https://github.com/jhonnerg/betterplancryptocoins.git,
 luego de esto se clonará todo el proyecto.

7.	Luego de clonar el proyecto abriremos una terminal con el comando Windows+R y colocaremos “CMD” y le daremos aceptar, luego
tenemos que ingresar donde se encuentre localizado la carpeta que clonamos el proyecto.
8.	Luego de acceder a nuestra carpeta ejecutaremos npm install –sabe

9.	Una vez instaladas las dependencias ejecutaremos ng serve para levantar la aplicación.

10.	Al terminar de compilar la aplicación iremos al navegador e ingresaremos a la URL siguiente: http://localhost:4200/ y podremos utilizar la aplicación web
